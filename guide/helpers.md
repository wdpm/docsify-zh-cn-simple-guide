# 文档助手

docsify 扩展了一些 Markdown 语法 。



## 强调内容

适合显示重要的提示信息，语法为 `!> 内容` 。

!>  一段重要的内容，可以和其他 **Markdown** 语法混用。 



## 普通提示

普通的提示信息，比如TODO 或参考内容等。语法为 `?> 内容` 。

?> TODO 内容



## 图片缩放

```
![logo](https://docsify.js.org/_media/icon.svg ':size=50x100')
```

![logo](assets/reimu.jpg ':size=191x108')



## 设置标题的 id 属性

```markdown
### 你好，世界！ :id=hello-world
```
